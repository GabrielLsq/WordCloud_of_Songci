## WordCloud_of_Songci
生成了宋词的词牌名和作者的词云。

使用[Wolfram Mathematica](http://www.wolfram.com/mathematica/?source=nav)编写。

宋词的数据库来自[chinese-poetry](https://github.com/chinese-poetry/chinese-poetry).
